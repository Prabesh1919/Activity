How to Apply to Your GitHub Account

  • 1

    Create a new empty Repository on GitHub

    Go to your GitHub profile and create an empty repository. Do not initialize it with a README, .gitignore, or license.

  • 2

    Configure and copy the command

    Use the simulator settings or painter canvas to craft your desired layout. Fill out the repository URL, then copy the generated command.

  • 3

    Run in terminal

    Open your terminal inside the script's project directory, and run the command. The script will automatically create a local folder, build the commit history, and push it to your remote repository.
